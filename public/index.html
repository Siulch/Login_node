Login API Documentation

Create User

POST https://loginsiulch.herokuapp.com/api/users

Create a new user.

Body: 

{
		"username": "paquito",
		"name": "paco", 
		"email": "paco@gmail.com",
		"password": "123456"
	}

Response:

{
			"success": true,
			"data": {
					"username": "paquito",
					"name": "paco",
					"email": "paco@gmail.com",
					"id": "eJqe7YVelBjUfy211E66T"
			},
			"message": "User created Correctly!"
	}
